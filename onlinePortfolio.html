Projects

This first project is 3.2.1, the food ordering program we did in Computer Science Essentials. It was completed on 3/5/24. This program allows the user to order food online. It is for the people who are too lazy to pickup the food from the restaurant, or are in a workspace like an office. This program was made using a dictionary for intial costs, and if/else statements for each food input the user wants. The main change I made to this program is changing the use of the while loops so the user can order more than one food item. I also defined totalCost and supersizeCost variable at the beginning of my code, making it easier to determine final cost and what the supersizing adds to your total. I think this program is a notable project because this is the first time where I had incoporated python concepts(if/else, loops, lists, etc.) and spent a lot of time thinking, debugging and facing challenges. A main challenge I had was not being able to show an error message with a loop when there was already a loop for ordering multiple items.
